# FriendFinder
## Tinder, but for friends.

### Summary
FriendFinder is a friend-dating app. Upon submitting the FriendFinder survey, FF's compatability calculator will match the user with their compatible friend!

### Features
* Mobile hamburger menu
* Modal pop-up window
* Node.js
* Express
* Heroku Deployment
